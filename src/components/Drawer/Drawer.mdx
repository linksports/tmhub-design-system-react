## Example

```typescript
<Drawer.Root>
  <Box style={{ height: "300px" }}>
    <Drawer.Trigger style={{ cursor: "pointer" }}>
      Drawer Trigger Click!
    </Drawer.Trigger>
  </Box>
  <Drawer.Overlay />
  <Drawer.Content
    sections={[
      {
        title: "セクションタイトル",
        items: [
          {
            label: "ホーム",
            icon: "Home",
          },
        ],
      },
      {
        items: [
          {
            label: "管理者",
            icon: "Admin",
          },
          {
            label: "検索",
            icon: "Search",
            badge: {
              text: "NEW",
              bold: true,
              type: "default",
            },
            selected: true,
          },
        ],
      },
    ]}
  />
</Drawer.Root>
```

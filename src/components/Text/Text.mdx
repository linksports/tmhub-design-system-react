# Text

## asChild

`Text` コンポーネントはレンダリングされるとHTMLの `p` エレメントになる。もし何らかの理由で要素を `span` にしたいなどあれば `asChild` 属性を使う。

```typescript
<Text asChild>
  <span>Sample Text</span>
</Text>
```

## wrap: 折り返し設定

`wrap` プロパティでは子要素の折り返しを指定する。  
初期値は `nowrap` で、折り返しは行なわずに `justify` の設定に従って子要素が並べられる。横幅が溢れる場合には、各子要素が均等に縮んで表示される。

```typescript
<Flex gap={2} width="400px" wrap="nowrap">
  <Box>Item 1</Box>
  ...
</FLex>
```

## align: 垂直方向の位置揃え

子要素の垂直方向の位置揃えを指定する。  
初期値は `stretch` で、子要素の垂直方向に余白があった場合には引き伸ばしてフィットさせる。

```typescript
<Flex gap={2} height="100px" align="stretch">
  <Box>Item 1</Box>
  ...
</FLex>
```

## alignCongtent: 折り返した際の垂直方向位置揃え

折り返しが行われた際の挙動になるので `wrap="nowrap"` の場合には無視される。

初期値は `stretch` で、子要素の幅を収まるように均等に引き伸ばして並べる。

```typescript
<Flex gap={2} height="100px" align="stretch">
  <Box>Item 1</Box>
  ...
</FLex>
```

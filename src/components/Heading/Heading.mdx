# Heading

## asChild

`Heading` コンポーネントはレンダリングされるとHTMLの `h1~h6` エレメントになる。もし何らかの理由で要素を `span` にしたいなどあれば `asChild` 属性を使う。

```typescript
<Heading asChild>
  <span>Sample Heading</span>
</Heading>
```

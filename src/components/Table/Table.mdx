# Table

## Props

### rows

Tableに表示される行データの配列。

### columns

Tableの列の設定用データの配列。

### firstFixedColumnCount

固定される列の数。最初のn列が固定されます。

### lastFixedColumnCount

固定される列の数。最後のn列が固定されます。

## Example

```typescript
type TestRow = {
  name: string;
  age: number;
};

const rows: TestRow[] = [
  { name: "TestName1", age: 20 },
  { name: "TestName2", age: 25 },
];

const columns: Column<TestRow>[] = [
  { key: "name", headerName: "Name" },
  { key: "age", headerName: "Age" },
];

return <Table rows={rows} columns={columns} />
```
